# SlabMod

A Latin font derived from [Slabo 13px/27px](https://github.com/TiroTypeworks/Slabo).

一款基于 [Slabo 13px/27px](https://github.com/TiroTypeworks/Slabo) 衍生的西文字体。

## Introduction / 字体简介

SlabMod is a modified version of Slabo, a slab-serif typeface designed specifically for legibility at small sizes on screen. This derivative retains the core characteristics of the original while making targeted adjustments to better suit the needs of Chinese typography and multilingual typesetting.

SlabMod 是基于 Slabo 的修改版本。Slabo 是一款专为屏幕小字号阅读而设计的粗衬线体（slab-serif）。本字体保留了原字体的核心特征，同时针对中文排版和多语言混排的需求进行了有针对性的调整。

## Modifications / 修改说明

### Lining Figures / 等高数字

The digit glyphs of Slabo are in **old-style** (text figures), with ascenders and descenders that break the alignment with uppercase letters and CJK ideographs. SlabMod replaces them with **lining figures** (aligning figures), ensuring consistent vertical alignment when used alongside Chinese, Japanese, and Korean characters.

Slabo 的数字为**旧式数字**（text figures），带有上下伸缩的笔画，与大写字母及 CJK 表意文字排列时无法对齐。SlabMod 将其替换为**等高数字**（lining figures），确保与中日韩文字混排时的垂直对齐。
![](https://github.com/lxgw/SlabMod/raw/main/Comparison.png)

### Hanyu Pinyin Support / 汉语拼音支持

In addition to the digit modification, SlabMod adds support for **Hanyu Pinyin letters** defined in GB/T 2312, including characters with tone marks such as `ǎ`, `ǒ`, `ǖ`, and more. This makes the font suitable for Chinese language education materials and pinyin annotations.

此外，SlabMod 还补充了 **GB/T 2312 标准中定义的汉语拼音字母**，包括带声调符号的字符，如 `ǎ`、`ǒ`、`ǖ` 等。这使得该字体适用于中文教学材料和拼音标注场景。

## Get the fonts / 获取字体

Go to the `TTF` folder in this repository to download the font files. WOFF/WOFF2 formats are also available in the `WOFF` and `WOFF2` folders.

进入本仓库的 `TTF` 文件夹获取字体文件。`WOFF` 和 `WOFF2` 文件夹中同时提供 WOFF/WOFF2 格式。
